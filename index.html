商業大廈渠喉更換 - 小型工程申報流程（中英對照）

  1. 現場勘查及技術評估 / Site Inspection & Technical Assessment
  2. 確認是否屬小型工程 / Confirm Minor Works Classification
  3. 委任註冊人士 / Appoint Registered Personnel
  4. 提交 Form MW03 / Submit Form MW03
    提交施工圖（包括去水圖）、方法、安全計劃等至屋宇署。
    Submit plans (including drainage layout), method statements, and safety plan.
  5. 等候備案 / Await Acknowledgement
  6. 關閉總渠及拆除 / Shut Down & Remove
  7. 安裝新渠喉 / Install New Pipe
